# C++编程简介

## 基本概念

### 基于对象（Object Based）

- class without pointer members (Ex: Complex)
- class with pointer members (Ex: String)


### 面向对象（Object Oriented）

- 继承（inheritance）
- 复合（composition）
- 委托（delegation）


## C++历史

- B语言（1969）
- C语言（1972）
- C++语言（1983）（new C -> C with class -> C++）


## C++演化

- C++ 98 (1.0)
- C++ 03 (TR1 - Technical Report 1)
- C++ 11 (2.0)
- C++ 14 (小改版)

学习C++需要学习C++语言本身以及C++标准库。
